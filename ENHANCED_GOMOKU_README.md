# 增强版五子棋游戏

这是一个功能丰富的五子棋游戏，提供了多种玩法模式和自定义选项，让传统五子棋游戏更加有趣和多样化。

## 功能特色

### 1. 游戏模式
- **双人对战模式**：两名玩家轮流下棋
- **人机对战模式**：与AI进行对战，支持三种难度级别

### 2. AI难度
- **简单**：AI随机选择位置下棋
- **中等**：AI使用策略算法进行对战
- **困难**：AI具有预判能力，可预测玩家的策略

### 3. 棋盘自定义
- 支持多种棋盘规格：9×9、13×13、15×15、19×19
- 不同规格适合不同水平的玩家

### 4. 获胜规则
- **五子连珠**：传统的五子连珠获胜规则
- **六子连珠**：六子连珠获胜，增加游戏挑战性
- **自定义连珠**：可设置需要N子连珠获胜（3-10子可选）

### 5. 增强功能
- **智能提示**：提供最佳落子位置建议
- **棋谱保存/加载**：可保存和加载游戏进度
- **游戏统计**：记录黑白双方步数和游戏时长
- **悔棋功能**：支持撤销上一步（人机模式下可撤销双方各一步）
- **计时功能**：记录游戏时长
- **星位标记**：在15×15和19×19棋盘上显示星位

## 游戏规则

1. 黑棋先行，双方轮流在棋盘上放置棋子
2. 率先在横、竖或斜方向上形成连续的N个同色棋子（N为获胜规则设定）的一方获胜
3. 可随时使用悔棋功能，增加游戏的容错率

## 操作说明

- 点击棋盘空白处放置棋子
- 使用游戏控制按钮进行相关操作
- 通过设置选项调整游戏参数

## 技术特点

- 使用Canvas绘制棋盘和棋子，界面美观
- 实现了复杂的AI算法，提供不同难度的挑战
- 响应式设计，适配不同屏幕尺寸
- 使用localStorage保存棋谱，数据持久化

## 文件结构

- `enhanced_gomoku.html`：游戏主界面
- `enhanced_script.js`：游戏核心逻辑
- `style.css`：样式文件
